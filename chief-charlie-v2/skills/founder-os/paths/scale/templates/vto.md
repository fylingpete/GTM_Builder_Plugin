# Vision/Traction Organizer™ (V/TO™)

> **Status:** empty_template — drafted in 1-2 Tage Off-Site, refresht jährlich (Annual Planning Session)
> **Team-required:** yes (full Leadership Team — solo-authored V/TOs sind structurally invalid)
> **Source:** EOS™ (Gino Wickman) — Vision Component, Tool #5

> **STRICT TWO-PAGE-DISCIPLINE.** Das V/TO™ ist deliberately zwei
> Seiten. Wenn Vision-Antworten länger werden, sind sie zu vage. Die
> Zwei-Seiten-Constraint macht Vagheit unmöglich zu verstecken. V/TO™
> wird FÜNFTES Tool installiert (nach Acc Chart → Rocks → Meeting Pulse
> → Scorecard) — nicht erstes. Vision-Session ohne Execution-Discipline
> produziert aspirational fluff, das niemand ownt.

## How to use this file

Das V/TO™ ist die Antwort auf 8 Fragen über wer die Company ist, wo
sie hingeht, und wie sie dorthin kommt. Page 1 ist Vision (stable
identity, slow-changing); Page 2 ist Traction (fast-changing execution).
Jedes Layer leitet sich vom Layer drüber ab — wenn die Kaskade
10-Year Target™ → 3-Year Picture™ → 1-Year Plan → Quarterly Rocks
nicht clean traceable ist, ist eine Schicht falsch.

Gebaut wird das V/TO™ in einem 1-2 Tage Off-Site mit dem FULL Leadership
Team. Solo-authored V/TOs sind strukturell invalid — Commitment ist
proportional zu Co-Authorship. Annual Planning Session refresht 3-Year
Picture™ + 1-Year Plan; Quarterly Planning refresht nur die Rocks.

## Rules (from the substance rubric)

- **two_pages_strict (non-negotiable):** Page 1 (Vision) + Page 2
  (Traction) — zwei Seiten total. Vagheit verstecken sich in Länge.
- **full_team_authored:** Komplettes Leadership-Team in Off-Site
  beteiligt. Solo-authored = invalid.
- **installed_after_foundations:** V/TO™ wird fünftes Tool installiert
  (nach Acc Chart, Rocks, Meeting Pulse, Scorecard) — nicht erstes.
- **cascade_traceable:** 10-Year Target™ → 3-Year Picture™ → 1-Year Plan
  → Quarterly Rocks: jede Schicht leitet sich von der darüber ab.
- **core_focus_two_parts:** Purpose/Cause/Passion + Niche — beide
  Sub-Parts non-omittable.
- **3yr_picture_concrete:** "Looks-Like"-Bullets sind specific genug,
  dass in 3 Jahren binär checkbar ist, ob sie eintrafen. Kein "world-
  class", kein "industry-leading".
- **rocks_3_to_7:** Quarterly Rocks pro Leader 3-7, single named owner,
  SMA-Kriterien, binary scoring.
