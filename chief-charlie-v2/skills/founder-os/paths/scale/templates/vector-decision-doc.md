# Vector Decision Document

> **Status:** empty_template — drafted at start of Step 3.4, locked when Vector picked + Core-Focus™-Filter angewandt
> **Team-required:** yes (Leadership-Team mit Visionary + Integrator + Function-Heads)
> **Source:** Skok Phase 11 (3 Vektoren + Sequencing) + Wickman Core Focus™ + YC Second Product

> **STRICT CORE-FOCUS-FILTER.** Jede Vector-Option läuft durch den
> Core Focus™-Filter VOR Financial-Analysis. Off-Cone-Vectors werden
> documented declined, nicht "interessant gemerkt". NDR > 110% sustained
> ist Precondition — unter 100% löst Vector-Add das falsche Problem.
> Maximal 1 Vector aktiv parallel — Sequencing schlägt Parallel.

## How to use this file

Phase 11b hat 3 valide Vektoren: Second Product, Vertical-Depth, M&A
(plus Geographic in 3.3). 3.4 picked GENAU EINEN Vector pro Decision-
Cycle. Multi-Vector parallel ist der typische Phase-11-Failure.

Das Doc dokumentiert: alle considered Vectors, Core-Focus™-Score pro
Vector, NDR-Precondition-Status, die finale Decision + Rationale, und
die documented Off-Cone Declines.

## Rules (from the substance rubric)

- **core_focus_first (non-negotiable):** Jede Vector-Option durch
  Core Focus™-Filter BEVOR Financial-Analysis. Off-Cone = declined.
- **ndr_precondition_check:** NDR > 110% sustained ≥ 4 Quartale ist
  Pre-Condition für Second-Product oder M&A. Unter 110% → Vector ist
  Geographic oder Vertical-Depth, niemals Second-Product.
- **one_vector_at_a_time:** Maximum 1 aktiver Vector parallel. Geographic
  (3.3) + Product-Line (3.4) nicht simultan.
- **same_three_properties_test (für Second Product):** Same ICP, Same
  Channel, Startup-inside-Startup. Wenn eines missing → declined.
- **10_customer_presell_required (für Second Product):** Vor Engineering
  müssen 10 Customers mit Real-Money pre-sellen. Customer-Asks ≠ Pre-Sells.
- **off_cone_documented_decline:** Jede Off-Cone-Opportunity, die
  declined wurde, wird documented mit Reason. Zeigt Discipline.
- **m_and_a_not_growth_shortcut:** M&A wird gleich treated wie Build-
  Vektor — Core-Focus™-Filter + NDR-Pre-Condition + Pre-Written-
  Integration-Milestones.
